# Joint

The anatomical location at which two or more bones make contact.

## Type Hierarchy

Joint is a type of AnatomicalStructure.

### Direct Subtypes

No types directly inherit from Joint.

## Properties

Joint defines the following properties in addition to those inherited from AnatomicalStructure:

- biomechnicalClass: The biomechanical properties of the bone.
- functionalClass: The degree of mobility the joint allows.
- structuralClass: The name given to how bone physically connects to each other.
