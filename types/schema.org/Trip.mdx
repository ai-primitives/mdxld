# Trip

A trip or journey. An itinerary of visits to one or more places.

## Type Hierarchy

Trip is a type of Intangible.

### Direct Subtypes

The following types directly inherit from Trip:

- BoatTrip: A trip on a commercial ferry line.
- BusTrip: A trip on a commercial bus line.
- Flight: An airline flight.
- TouristTrip: A tourist trip. A created itinerary of visits to one or more places of interest ([[TouristAttraction]]/[[TouristDestination]]) often linked by a similar theme, geographic area, or interest to a particular [[touristType]]. The [UNWTO](http://www2.unwto.org/) defines tourism trip as the Trip taken by visitors.
  (See examples below.)
- TrainTrip: A trip on a commercial train line.

## Properties

- itinerary: Destination(s) ( [[Place]] ) that make up a trip. For a trip where destination order is important use [[ItemList]] to specify that order (see examples).
- arrivalTime: The expected arrival time.
- offers: An offer to provide this item&#x2014;for example, an offer to sell a product, rent the DVD of a movie, perform a service, or give away tickets to an event. Use [[businessFunction]] to indicate the kind of transaction offered, i.e. sell, lease, etc. This property can also be used to describe a [[Demand]]. While this property is listed as expected on a number of common types, it can be used in others.
